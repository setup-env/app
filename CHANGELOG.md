# Changelog

All notable changes are documented here. The format follows

## [Unreleased]

### Added

- Cross-platform application foundation, configuration, diagnostics, and CI.
- Versioned module manifest and embedded official module catalog.
- Reusable schema-v1 system snapshot with static and JSON status output.
- Live terminal dashboard with bounded histories and network throughput.
- Reproducible Windows, macOS, and Linux release archives for amd64 and arm64.
- SHA-256 release checksums and minimal intentional archive contents.
- User-scoped PowerShell and POSIX bootstrap installers with explicit versions,
  verified upgrades, rollback, scoped uninstall, and offline tests.
- Isolated tag-triggered GitHub Release workflow and release-operations guidance.

No public version has been dated or declared here until its release is actually
published.
